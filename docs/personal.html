API - Personal

Erforderliche Parameter:

PersonalNr, Name1, Name2, Mitarbeitergrup, Token

Ausgeschlossene Parameter:

PKUUID_13

Optionale Parameter:

response, update

Konkretes Beispiel:

http://meinserver:8080/4DAction/api_put_employee?token=EDCD868258EA463D8608F0C06D52D391&response=JSON&Name1=Fred&Name2=Durst&Mitarbeitergrup=Auszubildender&PersonalNr=5050

In der JSON-Rückmeldung haben Sie beispielhaft folgenden Inhalt:

{"success":true,"error":"","wasUpdate":true,"PersonalNr":"5050","processingTime":8958,"errorMsg":""}

Methode: api_put_employeeHilfe
Hier den Host zum Profacto-Webserver eintragen
Hier das Token für Kunden-API einfügen
PersonalNr für den Mitarbeiter
Name1 des Mitarbeiters eintragen.
Name2 des Mitarbeiters eintragen.
Mitarbeitergruppe für den Mitarbeiter eintragen.
